# C-PROJECT-MANAGER
### CPROJECT-MANAGER is a custom useless build system made for fun, not recommended for production

## How to use 
- `git clone https://github.com/Sum1Code/C-PROJECT-MANAGER`
- `clang build.c -o builder`
- `./builder`
